# Apex Dental IV Sedation

Tablet-optimized clinical workflow app for moderate IV sedation procedures.
Single-file HTML application, runs entirely client-side. No PHI leaves the device.

## Current version
Apex-IV-Sedation-Syringe-v3.html

## Deployment
AirDrop the latest HTML to iPad → open in Safari → Share → Add to Home Screen.
